# Tour §2 — Values and bindings <span class="edition-badge tiny">tiny</span>

Jolt has two binding kinds:

| Syntax | Meaning |
| ------ | ------- |
| `$name` | Immutable binding — assign once |
| `$$name` | Mutable binding — reassign with `name = …` |
