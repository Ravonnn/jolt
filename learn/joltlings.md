# Joltlings exercises

Hands-on drills using the real `jolt` CLI. See the full guide in the repository:
[`tools/joltlings/README.md`](https://github.com/jolt-lang/jolt/blob/main/tools/joltlings/README.md).

```bash
cargo build -p jolt-cli -p joltlings
./target/debug/joltlings list
./target/debug/joltlings verify
```
